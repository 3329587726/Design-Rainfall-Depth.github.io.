## 📌 Project Overview
This project presents a rainfall depth analysis system designed to support hydrological studies and engineering design. It focuses on analyzing rainfall data and generating design rainfall depth under different return periods.

The application provides an intuitive interface for users to explore rainfall patterns, visualize data, and better understand extreme precipitation events.

---

## 🚀 Features
- 📊 Rainfall data analysis and visualization
- 🌧 Calculation of design rainfall depth
- ⏱ Support for multiple return periods
- 📈 Interactive charts for data interpretation
- 💡 User-friendly interface for easy exploration

---

## 🛠 Technologies Used
- Rstudio
- Data visualization libraries 
- GitHub Pages for deployment

## 🎯 Purpose
This project aims to:
- Assist engineers and researchers in hydrological analysis  
- Provide a simple tool for understanding rainfall intensity  
- Demonstrate practical data visualization skills
